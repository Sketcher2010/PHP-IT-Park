Реализовать класс Машина (Car) реализующий следующий интерфейс:
interface CarInterface {
    public function start();
    public function openDoor();
    public function closeDoor();
    public function setSpeed($number);
    public function run();
}

Где функции:
start - запуск двигателя машины
stop - остановка двигателя машины
openDoor - открытие двери машины
closeDoor - закрытие двери машины
setSpeed - включение какой-либо передачи (0 - нейтральная передача)
run - поехать на машине
break - торможение (остановка машины)

Условия:
Нельзя запустить двигатель с включенной передачей
Нельзя начать движение с выключенным двигателем
Нельзя начать движение на передаче не равной 1 или -1 (задняя)
Нельзя начать движение с открытой дверью
Нельзя выключить двигатель находясь в движении
Нельзя открыть дверь находясь в движении
Нельзя закрыть закрытую дверь
Нельзя открыть открытую дверь

Передачи:
-1 - задний ход
0 - нейтраль
1-5 - обычные передачи с 1 до 5
